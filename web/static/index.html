CSS Classes

This page shows the CSS class extensions we defined in loading UI. Bootstrap 4 provides lots of useful attribute-based classes, so we remove parts that could be customized by those classes from below definition.

Layout

Classes for size and position.

  • w-1200 - 1200px wide
  • w-1024 - 1024px wide ( lg size for bootstrap )
  • w-768 - 768px wide ( md size for bootstrap )
  • w-480 - 480px wide ( sm size for bootstrap )
  • h-gr - 61.8% tall ( golden ratio )
  • vh-75 - 75vh tall
  • vh-50 - 50vh tall
  • vh-25 - 25vh tall
  • vh-gr - 61.8vh tall ( golden ratio )
  • vertical-center - center any elements inside this element.
  • rwd - force block collapse when viewport is smaller.
  • rounded-xl - 1rem border radius
  • sticky - shorthand for position sticky. should custom top manually ( default 80px).
    demo as the scrollspy in the right or this page.
<div class="vertical-center"> <div> centered inside container </div> </div>

Aspect Ratio

Use aspect-ratio and one of following ratio to maintain a block's aspect ratio:

  • ratio-1by1 - 1:1
  • ratio-4by3 - 4:3
  • ratio-3by2 - 3:2
  • ratio-16by9 - 16:9
  • ratio-2by1 - 2:1

The block should be wrapped by another block which controls the width of it. div inside this block will be absolutely positioned and 100% in width ane height. Use text-center and vertical-center if you want to place content in the middle.

Separator

horizontal lines. Use spacing utility ( my-* ) to tweak gap between them.

  • sep - basic separator. identical to hr
  • sep-dashed - dashed separator
  • sep-light - light separator
  • sep-dark - dark separator
  • sep-text - separator with text.

Text

Classes related to text size and style.

  • text-sm - smaller in size ( .8em )
  • text-lg - bigger in size ( 1.25em )
  • text-xl - yet bigger in size ( 1.75em )
  • no-select - shorthand for user-select: none
  • clickable - shorthand for user-select: none
  • code-block - for showing codes
  • text-shadow - add standard size text shadow
  • text-shadow-sm - add small size text shadow
  • text-shadow-lg - add large size text shadow

Tips & Taglets

Hint user with tags and popups. Tooltips are re-implemented in pure css.

Tips

Short texts popped up when users hover or interact with certain element.

  • has-tips - indicate that there are tips in this element.
  • has-tips.tip-on - force tip to show
  • has-tips.manual - show tip only when tip-on is set.
  • has-tips.tip-sm - small tip
  • hover-tip - element to show when has-tips hovered or set tip-on
  • hover-tip.bottom - toward right. similar rules for right, left, top. default right if not specified
  • hover-tip.ghost - no mouse event on it. used for tips in list of has-tips elements.

Form & Button

Additional style and functions for forms and buttons.

  • btn-upload - upload button. auto hide the contained input element.

Background

Background style including transparency and pattern.

  • bg-semi-light- semi transparent light background./ rgba(255,255,255.8)
  • bg-semi-dark- semi transparent dark background./ rgba(0,0,0,.6)
  • bg-image - belt background. users still have to specify their own background-image.
  • bg-tile - tile background ( repeatable ). use along with bg-image.
  • bg-fixed - fixed background.
  • bg-portrait - centered, none repeated background.
  • bg-cover - background size: cover
  • bg-contain - background size: contain

bg-image + bg-tile

Use bg-image + bg-tile to add vivid background pattern

<div class="bg-image bg-tile" style="background-image:url(tile.png)"> </div>
